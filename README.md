# 42 School Cursus Projects

This repository contains my progress through the 42 School Cursus, from the C piscine through the Python milestone.

## 🧠 Projects Overview

### 🟦 Libft
Reimplementation of standard C library functions, plus additional utilities.
Focus: memory management, strings, linked lists.

### 🟩 ft_printf
Custom implementation of `printf`.
Focus: variadic functions and formatted output handling.

### 🟨 get_next_line
Function that reads a file line by line.
Focus: file descriptors, static variables, buffering.

### 🟪 push_swap
Algorithmic sorting project using two stacks and limited operations.
Focus: algorithms, complexity, optimization.

### 🐍 Python Milestone
A series of Python exercises and mini-projects (Python00–Python10) covering:
- Core syntax, functions, and exception handling
- OOP concepts: classes, abstract classes, factories, strategy pattern
- Functional programming: lambdas, map/filter/sorted, functools (reduce, partial, lru_cache, singledispatch)
- Decorators, closures, and scope
- Data validation with Pydantic
- Type checking (mypy) and linting (flake8)

### 🌀 A-maze-ing
Python project generating mazes from a config file, using bitmask-encoded walls, shortest path calculation, and maze visualization. Packaged as a pip-installable module.
