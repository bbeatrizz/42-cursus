## 🧠 Projects Overview

Reimplementation of standard C library functions, plus additional utilities.  
Focus: memory management, strings, linked lists.

---

### 🟩 ft_printf
Custom implementation of `printf`.  
Focus: variadic functions and formatted output handling.

---

### 🟨 get_next_line
Function that reads a file line by line.  
Focus: file descriptors, static variables, buffering.

---

### 🟪 push_swap
Algorithmic sorting project using two stacks and limited operations.  
Focus: algorithms, complexity, optimization.
